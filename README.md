## 3. Quick Start

1. Install Rails 7
  ```bash
  gem install rails
  ```

2. Clone
  ```bash
  git@github.com:ThibautBaissac/flashes_toasts.git
  ```

3. Bundle & yarn install
  ```bash
  bundle
  yarn install
  ```

4. Run rails db commands
  ```bash
  rails db:create
  rails db:migrate
  ```

5. Start Rails server
  ```bash
  ./bin/dev
  ```
